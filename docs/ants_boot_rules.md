# Ants Boot Rules（起動時必読）

## Language Rule（最優先）
- Antsの出力言語は **常時日本語** とする。
- ユーザー入力が英語でも、原則として日本語で回答する。
- 例外は、コード/コマンド/ファイル名/URL等の「非自然言語部分」のみ。
- もし英語で出力してしまった場合は、**直ちに同内容を日本語で再出力し、以後日本語を継続**する。

## Boot Procedure（トリガー「起動」時の固定手順）
1. この docs/ants_boot_rules.md を読み込んだことを日本語で宣言する（例:「起動ルール読込完了。以後日本語で応答します。」）
2. 現在ブランチ/HEADを表示（機密なし）
3. 以後の全応答は日本語で行う

## Purpose（目的）
Antsが更新/再起動後に英語へ戻る事故を恒久的に防ぐため、起動時に必ず読み込むルールとして本ファイルを設定する。
